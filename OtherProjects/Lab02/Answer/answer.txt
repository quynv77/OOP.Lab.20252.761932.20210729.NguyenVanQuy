Trả lời câu hỏi: 
Phần 8. Create Constructor method 
Ques1: - If you create a constructor method to build a DVD by title then create a constructor method to build 
a DVD by category. Does JAVA allow you to do this?

-> Trả lời: Không, Java KHÔNG cho phép.
Lý do:

- Hai constructor này đều chỉ có 1 tham số kiểu String
- Java phân biệt constructor dựa trên:
	+ Số lượng tham số
	+ Kiểu dữ liệu tham số
- Không phân biệt theo tên biến (title hay category)
Việc làm như vậy dẫn đến trùng chữ ký phương thức (method signature),làm Java không thể phân biệt các phương thức,
gây lỗi biên dịch.
